This file provides a template to publish a post based on R markdown including code chunks. If you want to publish the content of this R markdown on the website, follow these steps:

Publishing a post with Hugo

  1. set the working directory setwd()

  2. run blogdown::serve_site()

  3. push it via terminal to GitHub and the website
